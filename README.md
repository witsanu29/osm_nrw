# Village Health Volunteer Management System

ระบบจัดการรายชื่ออาสาสมัครสาธารณสุขประจำหมู่บ้าน (อสม.) เป็นระบบเว็บที่ใช้สำหรับจัดเก็บและบริหารจัดการข้อมูล อสม. ภายในพื้นที่หรือหน่วยงาน เพื่อให้สามารถค้นหา แก้ไข และส่งออกข้อมูลได้อย่างสะดวกและเป็นระบบ

ระบบถูกพัฒนาขึ้นเพื่อช่วยลดการจัดเก็บข้อมูลแบบเอกสาร และเพิ่มความสะดวกในการบริหารจัดการข้อมูล อสม. ในรูปแบบดิจิทัล

## ฟังก์ชันหลักของระบบ

### 1. เพิ่มข้อมูล อสม.

ใช้สำหรับเพิ่มข้อมูลอาสาสมัครสาธารณสุขเข้าสู่ระบบ เช่น

- ชื่อ
- นามสกุล
- หมู่บ้าน
- เบอร์โทรศัพท์
- ข้อมูลที่เกี่ยวข้องกับ อสม.

---

### 2. ดูรายชื่อ อสม.

สามารถดูรายการ อสม. ทั้งหมดในระบบในรูปแบบตารางข้อมูล พร้อมรองรับการแก้ไขและจัดการข้อมูล

---

### 3. ส่งออกข้อมูล Excel

ระบบรองรับการส่งออกข้อมูล อสม. เป็นไฟล์ **Excel** เพื่อใช้สำหรับ

- การรายงาน
- การสำรองข้อมูล
- การนำไปใช้งานในเอกสารอื่น

---

### 4. ออกจากระบบ

ผู้ใช้งานสามารถออกจากระบบได้อย่างปลอดภัยผ่านปุ่ม **Logout**

## โครงสร้างระบบโดยรวม

ระบบประกอบด้วยโมดูลหลักดังนี้

- ระบบเข้าสู่ระบบ (Authentication System)
- ระบบจัดการข้อมูล อสม.
- ระบบแสดงรายการข้อมูล
- ระบบส่งออกข้อมูล Excel
- ระบบ Dashboard

---

## วัตถุประสงค์ของระบบ

- จัดเก็บข้อมูล อสม. อย่างเป็นระบบ
- ลดการจัดเก็บข้อมูลแบบเอกสาร
- เพิ่มความสะดวกในการค้นหาและจัดการข้อมูล
- รองรับการส่งออกข้อมูลเพื่อการรายงาน
