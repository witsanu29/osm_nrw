# osm_nrw
Village Health Volunteer Management System  ระบบจัดการรายชื่ออาสาสมัครสาธารณสุขประจำหมู่บ้าน (อสม.) เป็นระบบเว็บที่ใช้สำหรับจัดเก็บและบริหารจัดการข้อมูล อสม. ภายในพื้นที่หรือหน่วยงาน เพื่อให้สามารถค้นหา แก้ไข และส่งออกข้อมูลได้อย่างสะดวกและเป็นระบบ  ระบบถูกพัฒนาขึ้นเพื่อช่วยลดการจัดเก็บข้อมูลแบบเอกสาร และเพิ่มความสะดวกในการบริหารจัดการข้อมูล อสม.
